# Family Costs Bot 2.0

Бот Telegram для учёта расходов (SQLite).

## Быстрый старт

1. Установите зависимости в виртуальном окружении:
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```
2. Создайте файл `.env` с переменными окружения (см. ниже).
3. Запустите приложение:
```bash
python main.py
```

## Переменные окружения
- `BOT_TOKEN` — токен Telegram-бота (обязательно)
- `DB_PATH` — путь к SQLite-БД (необязательно, по умолчанию `expenses.db`)
- `ALLOWED_USER_IDS` — список ID пользователей (через запятую) с доступом. Если пусто — доступ открыт всем.

Пример `.env`:
```env
BOT_TOKEN=your-telegram-bot-token
# DB_PATH=expenses.db
# Только указанные ID смогут пользоваться ботом (пример):
ALLOWED_USER_IDS=123456789,987654321
```

## Формат сообщений
Сообщение должно быть в формате: `<описание> <сумма>`

Примеры:
- `Кофе 3.5`
- `Такси 250`
- `Обед 12,40`

## Структура проекта
- `main.py` — запуск бота и обработчики
- `db.py` — работа с БД (инициализация, вставка)
- `strings.py` — текстовые константы
- `requirements.txt` — зависимости

## Разработка
- Правила форматирования: `.editorconfig`
